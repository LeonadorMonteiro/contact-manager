# Contact List Manager

A simple command-line contact manager built with Python.

## Features
- Add contacts
- View all contacts
- Search contacts by name
- Delete contacts
- Contacts are saved automatically to a JSON file

## How to use
Run the program:
python main.py

## Requirements
Python 3.x — no external libraries needed.
